# OS-PROJECT-4thsem-CPU-Scheduling
Project
🚀 CPU Scheduling Simulator (Java Swing)
📌 Project Description

The CPU Scheduling Simulator is a desktop-based application developed in Java using Swing GUI that demonstrates how different CPU scheduling algorithms work in an operating system. The system provides an interactive and visual way to understand process scheduling instead of only theoretical study.

Users can input process details such as Arrival Time, Burst Time, and Priority, then select a scheduling algorithm. The system simulates execution and generates a Gantt Chart, along with performance metrics like Waiting Time and Turnaround Time.

✨ Features
📊 Simulation of CPU Scheduling Algorithms:
First Come First Serve (FCFS)
Shortest Job First (SJF)
Priority Scheduling
Round Robin
📈 Gantt Chart Visualization
⏱️ Waiting Time & Turnaround Time Calculation
📉 Average Performance Metrics
🖥️ User-friendly Java Swing Interface
⚡ Step-by-step process execution
📄 PDF Report Generation (iText Library)
🎯 Objective

The main objective of this project is to help students visually understand CPU scheduling algorithms and analyze their performance in a practical and interactive environment.

🛠️ Technologies Used
Java (Core)
Java Swing (GUI)
AWT (Graphics)
Multithreading
iText PDF Library
